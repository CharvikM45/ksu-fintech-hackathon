How It Works

Four steps. Zero cloud required.

1

Network

A $35 Raspberry Pi broadcasts a local Wi-Fi hotspot, creating a sovereign microbank — no internet required.

2

Transactions

Users make real community transactions — P2P transfers, POS payments — through a standard smartphone browser.

3

TrustScore

Community trust builds the TrustScore — a reputation metric derived from consistent, verified local activity.

4

Fair Credit

A better TrustScore unlocks fairer credit through microfinance partnerships — true financial inclusion.

Features

Bank-Grade Offline Security

Every transaction is hash-chained and signed with secp256r1 ECDSA cryptography — the same math used in Bitcoin. Database-level locking with cryptographic UUIDs prevents double-spending even at the millisecond level.

[ Security Architecture ]

Vendor POS Mode

Merchants get a dedicated point-of-sale interface that generates offline QR codes for instant payment. Accept payments, issue receipts, and reconcile totals — bypassing expensive card fees and the internet entirely.

[ POS Interface Mockup ]

6-Layer AI Fraud Engine

A heuristic AI engine runs natively on the Raspberry Pi — no cloud API required. It analyzes transaction velocity, historical baselines, and social graph trust to catch anomalies and freeze fraudulent transfers instantly.

[ AI Fraud Engine ]

TrustScore Credit System

Every verified transaction builds a user's community TrustScore — a reputation metric that unlocks fairer access to microfinance lending. Better behavior means better credit, powered entirely by local activity data.
